| Typ tokenu | Wartości |
|------------|----------|
| KEYWORD | `if`, `while`, `for`, `print` |
| TYPE    | `int`, `str`, `double`, `bool` |
| BOOL    | `true`, `false` |
| ID      | identyfikatory |
| INT     | liczby całkowite |
| OP      | `+`, `-`, `*`, `/`, `=`, `>`, `<`, `!` |
| LBRACE  | `{` |
| RBRACE  | `}` |
| LPAREN  | `(` |
| RPAREN  | `)` |
| WS      | białe znaki |
